O que significa HTML?

(Voltar)

Criado por Tim Bernes-Lee, o HTML é uma linguagem, mas não de programação, e sim de marcação de texto, fornecendo uma estrutura básica que permite que os desenvolvedores Web possam criar e projetar páginas através de elementos, atributos e tags.

Estrutura do HTML

(Voltar)
  1. Declaração do Tipo de Documento (Document Type Declaration - DOCTYPE): Define a versão do HTML que está sendo usada no documento. Geralmente é a primeira linha do código HTML e informa ao navegador como interpretar o documento.
  2. Elemento 'html': É o elemento raiz de todo documento HTML. Todos os outros elementos HTML são descendentes deste elemento.
  3. Elemento 'head': Este elemento contém metadados e informações sobre o documento HTML, como o título da página, referências a scripts, estilos e outras informações.
  4. Elemento 'body': Contém o conteúdo principal do documento HTML que será exibido no navegador, como texto, imagens, vídeos, links e outros elementos.

O que são tags?

(Voltar)

De modo geral, as tags são marcações que vão informar a estrutura HTML da página. Toda tag tem um < e um> marcando seu início e fim.

Exemplos de tags

(Voltar)

Aqui estão algumas tags e suas funções:

  • title: É utilizado para modificar o texto que fica dentro da aba.

  • p: Inicia e termina um parágrafo.

  • a: Adiciona um hyperlink.

  • h1- h6: São utilizados para definir cabeçalhos, o número varia por importância, sendo o h1 o mais importante e o h6 o menos importante.

  • mark: O texto que estiver dentro dessa tag é marcado por outra cor.

  • small: Essa tag faz com que o texto tenha seu tamanho reduzido.

  • strong: O texto fica em negrito.

  • i: Essa tag faz com que o texto fique em itálico.

  • u: Deixa o texto sublinhado.

  • hr: Adiciona uma linha horizontal, normalmente é utilizada para separar seções de conteúdo.

  • ol: Lista ordenada.

  • ul: Lista não ordenada.

  • li: Item individual de uma lista.

  • blockquote: Utilizado para citações.

  • sub: Texto subscrito, posicionado abaixo da linha de base, como em elementos químicos.

  • sup: Texto sobrescrito, posicionado acima da linha de base, como em expoentes matemáticos.

  • del: Indica que um texto foi removido ou deletado do documento é exibido com uma linha através do texto.

  • abbr: É usado em abreviações ou acrônimos em documento HTML fornecendo informações quando o cursor paira sobre elas.
